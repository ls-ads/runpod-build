# Deployment Results

This directory contains the artifacts and results downloaded from RunPod deployments. 

Each deployment will create a subfolder named in the following format:
`{template_id}-{sanitized_gpu}-{timestamp}`

Examples:
- `qn96ymnjpd-nvidia-rtx-a4500-1708871234/`
- `n6m0htekvq-nvidia-geforce-rtx-4090-1708875678/`

Note: New results are appended as unique subfolders. This directory is not automatically cleared by the tool, allowing you to accumulate and compare artifacts from multiple builds.
